# Development Log - Archive Grammar

The website should not feel like a stack of interchangeable blog posts.

Scenes need literary air.
Cases need bureaucratic pressure.
Dossiers need interpretive sharpness.
World pages need reference clarity.

## Current Decision

- Let metadata drive presentation template.
- Keep Markdown as the source of truth.
- Build the archive so chamber material and bureau material can be read separately or together.
